Este projeto foi desenvolvido como atividade prática da disciplina de Fundamentos de programação WEB, do curso Análise e Desenvolvimentos de Sistemas da PUCPR.
Com o objetivo principal de ser um site de avaliação de jogos, para exercitar, de forma divertida, a escrita.
Funcionalidades: Cadastros de usuários, avaliações e comentários, armazenamento e consulta no banco de dados, uso o Bootstrap para estilização e responsividade.

Desenvolvi a primeira etapa somente com HTML e JavaScript.

Na segunda etapa, o projeto evolui e utilizei HTML+ PHP + CSS + Framework Bootstrap + SQL, utilizando XAMPP, para a conexão com o PHPMyAdmin.

IDE Utilizada: Microsoft VS Code.

Tecnologias Utilizadas: HTML5, JavaScript, CSS3, PHP, SQL(MySQL/MariaDB via PHPmyadmin), Bootstrap 4, XAMPP.

OBS: Para rodar o projeto localmente, é necessário ter o XAMPP instalado, iniciar o servidor Apache e MySQL, e importar o banco de dados via phpMyAdmin.
